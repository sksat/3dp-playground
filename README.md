# 3dp-playground

3Dプリント用のライブラリ・作品置き場。

| ライブラリ | 説明 |
|-----------|------|
| [dsub_panel_mount](dsub_panel_mount/) | D-SUBコネクタのパネルマウント |
| [heat_insert](heat_insert/) | ヒートセットインサート用穴 |
| [screw_post](screw_post/) | ネジポスト＋ナットポケット |
| [tamiya_parts](tamiya_parts/) | タミヤ楽しい工作シリーズ互換パーツ |
| [gridfinity](gridfinity/) | Gridfinity モジュラー収納システム |

各ライブラリの詳細は `library_name/README.md` を参照。
